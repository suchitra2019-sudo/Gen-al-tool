# Invoice Generator

A simple open-source GST invoice generator built with Flask. Generates invoices, stores data in Excel, and creates downloadable PDFs.

## Features
- Web form for invoice entry
- Stores invoice data in Excel for backup/tracking
- Generates PDF invoices
- Clean HTML/CSS design
- Download links for Excel and PDF

## Setup
1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/invoice-generator.git
   cd invoice-generator
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:
   ```bash
   python app.py
   ```
